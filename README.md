# 📋 bucket-list-react-native-test

> react-native 연습용 과제입니다.

### Install React-Native 
```cmd
npm install -g react-native-cli
npm install -g expo-cli
```

### Create Project
```cmd
expo init [project_name]
```

### Update Package & Dependencies
```cmd
npm install
```

### Run Project
```cmd
expo start
```
